Getting Started

Installing Statigen

Statigen can be installed from PyPI via Pip.

pip install statigen

Once the installation is complete, try the following command to see if it worked:

statigen --version

Starting a project

Create a directory where you want your page source and configuration files to live in, then create a .statigen.toml file and specify your site template and title.

$ mkdir mysite &amp;&amp; cd mysite
$ cat &gt; .statigen.toml &lt;<eof [site]="" [statigen]="" ```="" eof="" template="default/docs" title="My Site"> Statigen comes with the following templates out of the box:
&gt;
&gt; * [default/blog](getting-started/templates/blog.html)
&gt; * [default/docs](getting-started/templates/docs.html) (the one you are looking at right now)

Now create a bunch of `.md` files in the same directory. The section enclosed
in `+++` will be parsed as TOML and allows you to define properties in a file
that may be respected by the template.
